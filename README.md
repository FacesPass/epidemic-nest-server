## 初始化

```bash
$ npm install
```

## 启动项目

```bash
# 开发模式
$ npm run start

# 热更新模式
$ npm run start:dev

# 生产模式
$ npm run start:prod
```

## 测试

```bash
# 单元测试
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## 简介

使用 Nest.js 搭建相关的疫情小程序 API 服务支持

- 接入 Alibaba 疫情防控相关 API
